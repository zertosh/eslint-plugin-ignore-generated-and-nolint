# eslint-plugin-ignore-generated-and-nolint [![Build Status](https://travis-ci.org/zertosh/eslint-plugin-ignore-generated-and-nolint.svg?branch=master)](https://travis-ci.org/zertosh/eslint-plugin-ignore-generated-and-nolint)

Silently swallow [ESLint](https://eslint.org/) issues in files with `@generated` or `@nolint`.

## Install

```sh
$ npm install eslint-plugin-ignore-generated-and-nolint --save-dev
```

## Usage

Add `ignore-generated-and-nolint` to `plugins` in your ESLint config. That's it.
